# GitHubAchievements

A personal sandbox repo used to legitimately earn [GitHub profile achievements](https://github.com/ReconGrunt?tab=achievements) — no fake stars, no bot accounts, no third-party manipulation. Everything here is a real commit merged through a real pull request on a repo I own.

## What's earned here

| Achievement | How |
|---|---|
| Pull Shark | Opening + merging pull requests (`scripts/farm-achievements.ps1`) |
| Pair Extraordinaire | Merged commits carrying a `Co-authored-by:` trailer |
| Quickdraw | Opening and closing an issue within 5 minutes |

## Notes

- `progress.log` is a trivial append-only file — the content doesn't matter, it exists to give each PR a real diff.
- `scripts/farm-achievements.ps1` automates the branch → commit → PR → merge cycle. It's checked in for transparency, not because anyone else needs to run it.
- Achievements not attempted here: **Starstruck** (requires organic stars from distinct real users — not something to script) and **Public Sponsor** (a real GitHub Sponsors transaction, not repo-related).
